
# Mopedo Web tags README (2017-02-07)

The current tags directory contains tag2.js, the tag currently used in the Mopedo system.
For dynamic loading, use the following link (google bucket) instead of a GitHub link:

https://storage.googleapis.com/mopedo-web/tag2.js

... or for a minified version:

https://storage.googleapis.com/mopedo-web/tag2.min.js
